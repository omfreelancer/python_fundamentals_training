## Exercises (Optimized Order)

1. **Linear Search**  
   Write a function that returns:  
   - index of element if found  
   - or `-1` if not found  

2. **Counting Algorithms**  
   Write functions to:  
   - Count how many times a number appears in a list  
   - Count how many vowels are in a string  
   - Count how many uppercase letters are in a string  
   - Count duplicates in a list  

3. **Find Min and Max WITHOUT built-ins**  
   - Do not use `min()` or `max()`  

4. **Second Largest Number**  
   - Find the second largest number in a list  
   - Do it without sorting first  

5. **Reverse List (manual)**  
   - Reverse a list without using slicing (`lst[::-1]` is forbidden)  

6. **Bubble Sort**  
   - Implement bubble sort  
   - Focus on swap logic and nested loops  

7. **FizzBuzz**  
   - Classic logic test  

8. **(OPTIONAL) Binary Search**  
   - Implement binary search on a sorted list
